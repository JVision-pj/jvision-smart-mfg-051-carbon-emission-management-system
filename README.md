# 碳排放管理洞察站

盤查、計算與管理企業製造活動碳排放數據，支援永續報告與淨零目標管理之系統。

- 原始系統：碳排放管理系統（Carbon Emission Management System）
- 產業分類：智慧製造｜製造管理
- 使用部門：永續發展部門、廠務、財務、高階管理層
- 工作流程：排放源數據蒐集 → 排放量計算 → 目標比對分析 → 減碳對策執行 → 永續報告產出
- AI 應用：AI自動蒐集分散數據並估算排放量、智慧建議最具成本效益之減碳路徑
- 主要 KPI：總碳排放量、單位產品碳足跡、減碳目標達成率
- 代表產品：Schneider EcoStruxure Resource Advisor、IBM Envizi、Sphera
